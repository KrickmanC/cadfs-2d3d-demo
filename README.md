# CADFS — 2D в 3D

Русскоязычный интерактивный демонстрационный сайт по материалам статьи CADFS (CVPR 2026). Показывает концепцию конвейера от инженерного 2D-чертежа к параметрической истории FeatureScript и B-rep геометрии.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```

## Публикация на GitHub Pages

Push в ветку `main` автоматически запускает workflow `.github/workflows/deploy-pages.yml`. Сайт собирается с путём текущего репозитория и публикуется через GitHub Pages.

Важно: загрузка PDF в текущей версии является интерфейсной симуляцией. Исследование CADFS использует текст или четыре 2D-рендера, а распознавание производственных PDF-чертежей показано как отдельный продуктовый слой.
